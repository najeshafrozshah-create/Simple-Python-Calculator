# Basic Python Calculator

This is a simple command-line calculator written in Python.  
It allows the user to perform the following operations:

- Addition
- Subtraction
- Multiplication
- Division

---

## How it works

1. The program asks the user to input two numbers.
2. Then it asks what operation they want to perform.
3. Based on the input (sum, add, +, minus, subtraction, multiply, *, divide, etc.), it performs the calculation and prints the result.

---

## Example Run

Enter the 1st No. : 10
Enter the 2nd No. : 5
what you want to do? add

The Sum is 15.0
